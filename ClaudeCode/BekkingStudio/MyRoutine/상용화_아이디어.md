## 미구현 — 향후 로드맵

### 7. AI 인사이트 ⭐ (다음 우선순위)
- 패턴 분석: "화요일에 유동 시간이 가장 많습니다"
- 상관관계: "운동 후 몰입 시간이 평균 30% 증가합니다"
- 주간 AI 리포트 자동 생성
- 통계 대시보드 인사이트 카드 ("전주 대비 +15%")
- Claude API 또는 로컬 분석 알고리즘으로 구현
- **난이도**: 중 | **임팩트**: 높음 (차별화 요소)

### 8. 홈 화면 위젯 / 원탭 기록
- 홈 화면 위젯에서 앱 안 열고 바로 활동 기록
- 현재 활동 & 경과 시간 위젯 표시
- PWA에서는 제한적 → 네이티브 앱(React Native 등) 전환 시 구현
- **난이도**: 높음 (네이티브 필요) | **임팩트**: 높음

### 9. 추가 내보내기 형식
- CSV/Excel 내보내기 (데이터 분석용)
- Google Calendar 연동
- **난이도**: 낮음 | **임팩트**: 중

### 10. 푸시 알림
- 목표 미달성 리마인더 (현재는 앱 내 배너만)
- 하루 마감 알림 ("오늘 기록을 마감하세요")
- PWA Push API 또는 네이티브 전환 시
- **난이도**: 중 | **임팩트**: 중

### 11. 커뮤니티 & 소셜 확장
- 친구와 달성률 비교 (선택적 옵트인)
- 루틴 템플릿 공유 ("직장인 루틴", "학생 루틴")
- 비슷한 목표를 가진 사람들 그룹
- **난이도**: 높음 | **임팩트**: 중 (바이럴 마케팅)
